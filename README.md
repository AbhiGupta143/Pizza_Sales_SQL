# Pizza_Sales_SQL

Pizza Sales SQL Analysis Project
An end-to-end SQL project analyzing pizza sales data to extract actionable business insights. Performed order, revenue, and pricing analysis, identified top-selling pizzas, popular sizes, and high-revenue categories. Used joins, aggregations, and window functions to analyze hourly order trends, daily averages, cumulative revenue, and category-wise performance. Delivered insights such as top pizzas by quantity and revenue, revenue contribution percentages, and sales growth over time to support data-driven decision-making.
